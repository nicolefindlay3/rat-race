🔗 Connect Database

One-time setup · takes about 5 minutes · completely free

1
Create a free Supabase account
Go to supabase.com → click Start your project → sign up with GitHub or email. It's free — no credit card needed.
2
Create a new project
Click New project → choose your organisation → name it family-hub → set any database password → choose the region closest to you → click Create new project.
Wait about 60 seconds for it to finish.
3
Run this SQL to create the data table
In your project, click SQL Editor in the left menu → click New query → paste the code below → click Run.
create table if not exists app_state ( id integer primary key default 1, data jsonb default '{}', updated_at timestamptz default now() ); insert into app_state (id, data) values (1, '{}') on conflict do nothing; alter table app_state enable row level security; create policy "public_access" on app_state for all using (true) with check (true);
4
Copy your Project URL and anon key
Go to Project Settings (gear icon, bottom-left) → API.
You'll see:
• Project URL — looks like https://xxxx.supabase.co
• Project API keys → anon / public — a long string
Paste both below.
5
Paste your details and connect

Your credentials are saved only on this device. All family data is stored in your own Supabase database.

⚠️
This app must be hosted online (Netlify). Google won't allow sign-in from a local file.
1
Create a Google Cloud project
Go to console.cloud.google.com → top dropdown → New Project → name it "Family Hub" → Create.
2
Enable Google Calendar API
APIs & Services → Library → search Google Calendar API → Enable.
3
OAuth consent screen
APIs & Services → OAuth consent screen → External → Create → fill in app name + your email → Save & Continue through all steps → Back to Dashboard → under Test users click + Add Users → add your Google email.
4
Create credentials
APIs & Services → Credentials → + Create Credentials → OAuth client ID → Web application → under Authorised JavaScript origins add your Netlify URL → Create → copy the Client ID ending in .apps.googleusercontent.com
5
Paste Client ID and connect
